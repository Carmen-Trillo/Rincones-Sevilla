## ğŸ’»How to install this project:

1. Clone the project
```bash
git clone https://github.com/Carmen-Trillo/Rincones-Sevilla
```

2. Install dependencies
```bash
npm install
```

3. Install React Router
```bash
npm install react-router-dom@6
```   

4. Install Bootstrap
```bash
npm install react-bootstrap bootstrap
```

5. Install React Pginate
```bash
npm install react-paginate 
```
6. Install API
```bash
npm install -g json-server
```  
6. Install Axios
```bash
npm install axios
```  
8. Install React Form
```bash
npm install react-hook-form 
```
9. Install Contact Form
```bash
npm install emailjs-com 
```
9. Install Test
```bash
npm install --save-dev jest
npm install --save-dev babel-jest @babel/core @babel/preset-env
npm install --save-dev ts-jest
```
10. Activate the server and keep this terminal open
```bash
npm run dev
```   
11. Activate API
```bash
json-server --watch ./api/product.json
```
